## 📌 Descripción

Patitas Felices es una maqueta web estática desarrollada con **HTML5 y CSS3 puro**, cuyo objetivo es ampliar la presencia digital de la fundación y dar a conocer su labor de rescate, cuidado y adopción de animales.

El proyecto transmite la identidad de la organización y permite a los visitantes:

- Conocer la misión y visión.
- Ver animales disponibles para adopción.
- Acceder a perfiles individuales de cada animal.
- Encontrar información de contacto.

Todo el proyecto fue desarrollado **sin JavaScript**, utilizando únicamente HTML y CSS, incluyendo simulaciones de interacción visual.

---

## 🛠 Tecnologías utilizadas

- HTML5
- CSS3
- Flexbox
- Media Queries (Responsive Design)
- Animaciones CSS (@keyframes)
- Pseudoclases CSS (:hover, :focus, :placeholder-shown)

## 📄 Secciones Implementadas

### 🏠 Página Principal (pagina.html)

Contiene:

- Banner principal con nombre de la fundación
- Slogan representativo
- Botón de llamada a la acción
- Sección informativa sobre la fundación
- Diseño atractivo y jerarquía visual clara
- Adaptación responsive

---

### 🐶 Catálogo de Animales

Se muestran animales disponibles mediante tarjetas individuales.

Cada tarjeta incluye:

- Imagen
- Nombre
- Edad aproximada
- Tamaño o raza
- Estado (Disponible / Adoptado)
- Botón para ver perfil completo

---

### 🐕 Perfil de Animal

Cada archivo de tarjeta representa el perfil detallado de un animal e incluye:

- Imagen ampliada
- Descripción (historia, comportamiento, salud)
- Características generales
- Botón “Solicitar adopción” (maqueta visual)

## 🎨 Diseño y Estilos

El archivo `style1.css` contiene:
